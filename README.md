# "Enmity" Amongst "Countrymen" — Replication Materials

This repository contains the data and replication code for “Enmity” Amongst “Countrymen”: Transnational Elite Cueing, the Turkish Migrant Electorate, and Erdoğan’s 2017 Anti-Endorsement in Germany, which examines the electoral effects of Erdoğan's August 2017 anti-endorsement of three major German parties on Turkish-origin voters in North Rhine-Westphalia using a spatial difference-in-differences design.

---

---

## Scripts

### `mainworkbook.R`
> **Run this first.**

Processes the raw federal election data, merges it with demographic covariates, and generates the primary descriptive and analytical plots used in the thesis. Also produces the cleaned intermediate data objects read in by the NRW and Cologne analysis scripts. Running the subsequent scripts without first running this file will produce errors.

---

### `nrwworkbook.R`
Contains the difference-in-differences models for the North Rhine-Westphalia Gemeinde-level analysis, including:
- Aggregated anti-endorsed party models
- Party-specific models
- Placebo analysis
- Associated output tables

*Depends on objects generated by `mainworkbook.R`.*

---

### `kolnworkbook.R`
Contains the Stimmbezirk-level analysis for Cologne, including the party-specific DiD models and robustness checks at the finer geographic resolution.

*Depends on objects generated by `mainworkbook.R`.*

---

## Usage

1. Clone the repository and open the project in R or RStudio
2. Adjust the working directory at the top of each script to match your local file path before running anything — each script contains specific calls.
3. Run `mainworkbook.R` in full before running either of the other two scripts
4. `nrwworkbook.R` and `kolnworkbook.R` can then be run independently in any order

---

## Data

The cleaned datasets in the `data/` folder include *Zweitstimme* party-list vote shares from the 2017 NRW state and federal elections merged with demographic covariates from the German census. A full description of variable construction, sources, and cleaning decisions is available in the thesis itself.

---

## Dependencies

All scripts are written in R. Key packages include:

| Package | Purpose |
|---|---|
| `fixest` | Difference-in-differences estimation |
| `modelsummary` | Regression table output |
| `ggplot2` | Visualisation |
| `tidyverse` | Data manipulation |
| `sf` | Spatial data handling |
| `ggcorrplot` | Correlation matrix plots |

Package versions can be confirmed by running `sessionInfo()` after loading the scripts.
